# Task API! 

## Setup
- Install the dependencies
```bash
bun install
```
- Create a `.env` file in the root directory (see `.env.example` for an example)
- Generate migration files
```bash
bun run db:generate
```
- Run the migrations
```bash
bun run db:migrate
```

## Development
To start the development server run
```bash
bun run dev
```
